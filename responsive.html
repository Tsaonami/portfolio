Responsive
Layout

July 14, 2017

This page's web layout changes mainly by resizing. As the screen shrinks through width, the images shrink along with the viewport, however the icons and text remains the same sizes. This allows the interface to maintain legibility. Also, once the screen shrinks to a specific size, the navigation bar on the bottom changes in size and interactivity. The bars turn from part of the regular screen sized bars to bars that extend across the shrunken screen. This provides a comfortable user experience especially for this on mobile phones or who need to arbitrarily minimize the screen. Also the fuze logo picture relocates itself to a more convenient yet still accessible location before the footer.
